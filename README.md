# OpenID Authentication
Authentication with OpenID
## Quick start
add ```.env``` file in ```openid-auth``` folder
```dotenv
    CLIENT_ID=z...s
    CLIENT_SECRET=TQ.........dany3
```
## Commands:
```shell
npm install --save @nestjs/passport passport
npm i openid-client
npm i --save-dev @types/passport
npm i @nestjs/config
```
